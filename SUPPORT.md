# Support

If you need help with SwiftCause, use the right channel so questions reach the right place and issues stay actionable.

## Where To Get Help

### GitHub Discussions

Use GitHub Discussions for:

- setup help
- usage questions
- architecture questions
- contribution guidance
- early ideas and product discussion

This should be the default place for most support questions.

### Issue Tracker

Use GitHub Issues only for:

- confirmed bugs
- clearly defined feature requests
- specific actionable tasks

Please do not open Issues for general support or how-to questions.

### Security Reports

If you believe you found a security issue, do not post it publicly in Discussions or Issues.

Follow the guidance in [SECURITY.md](./SECURITY.md).

## Before Asking For Help

Please check:

- [README.md](./README.md)
- [CONTRIBUTING.md](./CONTRIBUTING.md)
- [ROADMAP.md](./ROADMAP.md)
- [CHANGELOG.md](./CHANGELOG.md)
- architecture and implementation docs in [docs](./docs)

When asking for help, include:

- what you were trying to do
- what happened instead
- steps to reproduce
- screenshots or logs if relevant
- your local environment details

## Maintainers

If additional support channels such as Discord, email, or community chat are introduced later, add them here and keep this document up to date.
